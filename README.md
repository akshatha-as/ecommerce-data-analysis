# 📊 E-Commerce Sales Dashboard

## Overview

This project presents an interactive **Power BI E-Commerce Sales Dashboard** designed to analyze sales performance, profitability, customer segments, and product trends. The dashboard helps stakeholders monitor key business metrics and identify opportunities to improve revenue and profit.

---

## Problem Statement

E-commerce businesses generate large amounts of transactional data every day. Without proper analysis, it becomes difficult to answer important business questions such as:

- Which products contribute the most to revenue?
- Which products are underperforming?
- Which regions generate the highest sales?
- How does shipping type impact sales?
- Which customer segments drive business growth?
- What is the overall profit margin?

The objective of this project is to transform raw sales data into meaningful insights using Power BI.

---

## Project Objectives

- Track Year-to-Date (YTD) sales performance.
- Monitor profit and profit margin.
- Analyze sales quantity trends.
- Identify top-performing and low-performing products.
- Compare sales across different regions and states.
- Understand customer segment distribution.
- Evaluate sales by shipping type.

---

## Dashboard KPIs

The dashboard provides the following key performance indicators:

- **YTD Sales**
- **YTD Profit**
- **YTD Quantity**
- **YTD Profit Margin**

---

## Dashboard Analysis

### 1. Product Performance Analysis

The dashboard highlights:

- Top 5 products by YTD sales.
- Bottom 5 products by YTD sales.
- Product categories contributing the most revenue.

This helps businesses focus on high-performing products and optimize inventory for low-performing items.

---

### 2. Regional Analysis

The dashboard includes:

- Sales by region.
- Sales by state (map visualization).

This analysis helps identify strong and weak markets and supports location-based business strategies.

---

### 3. Customer Segment Analysis

The dashboard allows users to explore sales performance across different customer segments.

Business teams can use these insights to:

- Improve customer targeting.
- Personalize marketing campaigns.
- Increase customer retention.

---

### 4. Shipping Analysis

The dashboard compares sales across shipping methods to understand:

- Customer preferences.
- Operational efficiency.
- Distribution patterns.
